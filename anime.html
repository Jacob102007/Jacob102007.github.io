Introducción a HTML

HTML es el lenguaje de marcado que define la estructura y el contenido de una página web. Cada elemento en una página web, como textos, imágenes, enlaces y formularios, se define utilizando etiquetas HTML. Estas etiquetas indican al navegador cómo debe presentar el contenido. Aunque HTML por sí solo solo estructura el contenido, a menudo se combina con CSS (para estilos) y JavaScript (para interactividad) para crear sitios web completos y funcionales.

Estructura Básica de una Página HTML

Para comenzar, necesitas entender la estructura básica de un documento HTML. Un documento HTML típico se organiza en una estructura jerárquica que incluye elementos clave como el encabezado, el cuerpo del documento y varios tipos de contenido.


<!DOCTYPE html>
<html>
    <head>
        <title>Mi Primera Página</title>
    </head>
    <body>
        <h1>Hola Mundo</h1>
        <p>Esto es una página web simple.</p>
    </body>
</html>

Introducción a CSS

CSS se usa para dar estilo a las páginas web. En este tutorial aprenderás a aplicar estilos básicos con CSS.


body {
    font-family: Arial, sans-serif;
    margin: 20px;
}

h1 {
    color: #333;
}
